UC-02: Photo Upload for Style Analysis

## Description
User uploads a photo of themselves so the AI can analyze body type and current style.

## Actors
Primary: User
Secondary: AI Analysis Service, System Storage

## Preconditions
User must be logged in to the system.

## Main Flow
1. User navigates to Dashboard
2. User clicks "Upload Photo" button
3. User selects JPG or PNG image from device, under 5MB
4. System uploads and stores the image
5. System runs AI analysis on the photo
6. System displays "Analysis Complete" confirmation to user

## Alternative Flows
A1: File too large → System shows "File must be under 5MB" error
A2: Wrong file type → System shows "Only JPG/PNG allowed" error
A3: Upload fails → System shows "Upload failed, try again" error

## Postconditions
User's photo is stored and body type/style data is saved for recommendations.
