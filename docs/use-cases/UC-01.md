## Description
User registers and logs into the Cloth AI Outfit Recommender system.

## Actors
Primary: User
Secondary: System Database

## Preconditions
User does not have an account OR user has an account and wants to log in.

## Main Flow
1. User navigates to Registration/Login page
2. User enters email and password
3. System validates credentials
4. System creates account or logs user in
5. System redirects user to Dashboard

## Alternative Flows
A1: Invalid email/password → System shows error message
A2: Email already exists → System asks user to log in instead

## Postconditions
User is authenticated and can access system features.
