# UC-05: Share Outfit to Social Media

## Description
User shares a saved outfit recommendation to Instagram to get feedback from friends.

## Actors
Primary: User
Secondary: Instagram API, System

## Preconditions
1. User must be logged in
2. User must have a saved outfit from UC-04

## Main Flow
1. User navigates to "Favorites" page
2. User clicks on a saved outfit
3. User clicks "Share to Instagram" button
4. System generates an image/card with the outfit description
5. System opens Instagram share dialog with the image pre-loaded
6. User posts the outfit to Instagram

## Alternative Flows
A1: Instagram not connected → System prompts users
