# UC-03: Get AI Outfit Recommendations

## Description
User requests AI-generated outfit recommendations based on occasion, weather, and their style analysis.

## Actors
Primary: User
Secondary: AI Recommendation Engine

## Preconditions
1. User must be logged in
2. User must have completed photo upload and style analysis from UC-02

## Main Flow
1. User navigates to "Get Recommendations" page
2. User selects occasion: e.g. Work, Casual, Party
3. User selects weather: e.g. Sunny, Rainy, Cold
4. User clicks "Get Outfit Ideas" button
5. System sends user data and selections to AI Recommendation Engine
6. System returns 3 text-based outfit recommendations with clothing item descriptions
7. Recommendations are displayed to user within 10 seconds

## Alternative Flows
A1: No photo uploaded → System prompts user to complete UC-02 first
A2: AI service unavailable → System shows "Service temporarily unavailable" error

## Postconditions
User receives 3 personalized outfit recommendations they can view, save, or share.
